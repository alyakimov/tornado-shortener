Описание API:

Данный сервис имеет свой API, Для того чтобы воспользоваться им достаточно передать переменную url в GET запросе по адресу {{ options.domain_name }}/_, например так:

$ curl -XPOST -d "uri=http://google.com" {{ options.domain_name }}/_

В теле ответа будет возвращен сокращенный урл: http://{{ options.domain_name }}/ab

Статистика:

Для доступа к статистике добавьте к короткому урлу знак "+", например так:

{{ options.domain_name }}/ab+
